1. AUTOMATIC PLUGIN INSTALLATION (PREMIUM)
===========================================

**FREE VERSION:**
- Users see list of required/recommended plugins
- Click links to WordPress.org
- Manually install and activate each plugin
- Return to wizard to continue

**PREMIUM VERSION:**
- Users see list of required/recommended plugins
- Click "Install Plugins" button
- Automatic installation and activation
- Wizard continues automatically

Implementation:
- Method: Conjure_Freemius::can_auto_install_plugins()
- Returns: false (free), true (premium with valid license)
- Location: Plugin step in wizard

4. THEME PLUGIN BUNDLING (NEW FEATURE)
=======================================

Theme developers can now bundle plugins directly with their theme using a simple JSON configuration file!

### How It Works:

1. Create folder in theme: `/conjurewp-plugins/`
2. Add configuration file: `plugins.json`
3. Place premium plugin ZIPs in folder
4. ConjureWP auto-discovers and registers them

### Features:

‚úÖ Auto-discovery - no code required
‚úÖ JSON configuration - simple and clean
‚úÖ Support for free (WordPress.org) plugins
‚úÖ Support for premium/bundled plugins (ZIP files)
‚úÖ Mandatory vs Optional tagging
‚úÖ Works for ALL demos automatically
‚úÖ Combines with demo-specific plugins

### Example Configuration:

```json
{
  "plugins": [
    {
      "name": "Contact Form 7",
      "slug": "contact-form-7",
      "mandatory": true,
      "description": "Required for contact forms"
    },
    {
      "name": "Elementor Pro",
      "slug": "elementor-pro",
      "file": "elementor-pro.zip",
      "mandatory": true,
      "version": "3.16.0",
      "description": "Premium page builder (bundled)"
    },
    {
      "name": "Yoast SEO",
      "slug": "wordpress-seo",
      "mandatory": false,
      "description": "Recommended for SEO"
    }
  ]
}
```
