HOW IT WORKS
============

FREE VERSION WORKFLOW:
1. Run setup wizard
2. Select demo (if multiple available)
3. Generate child theme (optional)
4. SEE LIST of required plugins
5. MANUALLY install plugins from WordPress.org
6. Return to wizard
7. Import demo content, widgets, customizer
8. Complete setup

PREMIUM VERSION WORKFLOW:
1. Run setup wizard
2. Select demo (if multiple available)
3. Generate child theme (optional)
4. SEE LIST of required plugins
5. CLICK "Install Plugins" button
6. Plugins automatically installed & activated
7. Import demo content, widgets, customizer
8. Complete setup

The difference? Steps 5-6 are automated in Premium.

TECHNICAL IMPLEMENTATION
=========================

The plugin uses Freemius for licensing and deployment:

1. DEPLOYMENT
   - Upload full plugin to Freemius dashboard
   - Freemius automatically generates TWO versions:
     a) FREE version (WordPress.org compatible)
     b) PREMIUM version (with Freemius SDK)

2. CODE STRUCTURE
   - Premium code wrapped in @freemius:premium-start/end tags
   - Free version: Premium code automatically stripped
   - Both versions work perfectly in their environments

3. FEATURE GATING
   - Conjure_Freemius::can_auto_install_plugins()
   - Returns FALSE in free version
   - Returns TRUE in premium version (with valid license)

FREQUENTLY ASKED QUESTIONS
===========================

Q: Can I still import demos with the free version?
A: Yes! All demo import features work in the free version. You just 
   need to manually install the required plugins first.

Q: What happens if my premium license expires?
A: The plugin continues to work, but automatic plugin installation 
   will be disabled. Manual installation still works.

Q: Can I upgrade from free to premium?
A: Yes! Purchase a license and activate it. Your existing setup 
   remains intact.

Q: Does premium include theme licenses?
A: No. ConjureWP is separate from theme licensing. This is a plugin 
   license for automatic plugin installation features.

Q: Can I use premium on multiple sites?
A: Depends on your license tier (single, 5-site, or unlimited).

Q: What about theme developers?
A: Purchase lifetime integration and ALL your theme users get 
   automatic plugin installation without needing their own license.
